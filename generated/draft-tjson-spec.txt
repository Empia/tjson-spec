   Tagged JavaScript Object Notation (TJSON) Data Interchange Format
                            draft-tjson-spec

Abstract

   Tagged JavaScript Object Notation (TJSON) is a JavaScript Object
   Notation (JSON) [RFC7159] compatible data interchange format with
   first-class type annotations ("tags") for supporting a richer set of
   types within JSON documents.
